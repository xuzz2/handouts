Power Automate Use Cases

1. Approval Flow - Leave Request

Scenario: Automate employee leave request approvals

Steps to Build:

  1. Create trigger: "When a new response is submitted" (Forms/SharePoint)
  2. Add "Start and wait for an approval" action
  3. Configure approver: Direct manager (dynamic from HR system)
  4. Set timeout: 24-48 hours for response
  5. Based on approval result:
    • Approved: Send confirmation to employee & update calendar
    • Rejected: Notify employee with reason
    • Timeout: Escalate to secondary approver

Time saved: ~15 minutes per request

2. CLM Report Email Notification

Scenario: Auto-send updated reports to stakeholders

Trigger: Scheduled (daily/weekly) or when report is updated

Actions:

  1. Get report from specified location (SharePoint/OneDrive)
  2. Check if data has changed since last run
  3. Format email with key metrics highlighted
  4. Send to distribution list
  5. Log sent report for audit trail

3. New Hire Onboarding List Flow

Scenario: Scicom automated onboarding workflow

Process flow:

  1. HR adds new hire to master list ‚Üí Triggers flow
  2. Create user accounts (Active Directory/Office 365)
  3. Assign software licenses & security groups
  4. Setup SharePoint access & team memberships
  5. Send welcome email with credentials and first-day instructions
  6. Notify IT, manager, and facilities team

Benefits: Reduces manual setup from 2 hours to 15 minutes

Copilot Prompt Library

Email Prompts

"Rewrite this message as a professional business email: [paste your draft]"
"Summarize this email thread highlighting key decisions and action items"
"Draft a follow-up email to a client about project [project name] after our meeting on [date]"

Document & Report Prompts

"Identify 3 key insights from this dataset and explain them in simple terms"
"Convert these meeting notes into a structured summary with: decisions, action items, owners, deadlines"
"Improve the clarity and professionalism of this project status report"

Creative & Planning Prompts

"Brainstorm 5 agenda items for a team meeting about [topic]"
"Suggest 3 different approaches to solve [problem statement]"
"Create a checklist for [process name] with estimated time for each step"
